HyperText Markup Language (HTML)
This is the main type of document on the web.

HTML is made up of:
text content (what you see)
markup (what it looks like)
references to other documents (e.g. images, videos)
links to other pages

HTML markup
Opening tag, contents, closing tag
<NAME>Contents</NAME>
Element

b tag
makes text bold
<b>contents</b>

em tag
makes text italicized
<em>contents</em>

HTML Attributes
<TAG ATTR="value">contents</TAG>

anchor tag
makes text a link
<a href= "url"contents</a>

image tag
inserts an image

<img src="url" alt="text">
note: this is known as a "void tag", i.e. no contents, no closing tag

br tag
creates a
line break
<br>
note: this is an "inline" element

p tag
"paragraph" creates a

text box

<p>contents</p>
note: this is a "block" element

span tag
contains text inline
<span>contents</span>

div tag
contains
text
in a block
<div>contents</div>>

HTML-CSS-DOM
HTML - HyperText Markup Language - the standard markup language used to create web pages.
CSS - Cascading Style Sheets - style sheet language used for describing the look and formatting of a document written in a markup language.
DOM - Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.
